## 📦 Детальная схема карточки сценария

### 📌 Состояние: СВЁРНУТО

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  🎬 "Как стать миллионером за 30 дней"              │
│  📅 9 ноября 2025, 14:30    🎞️ Сцен: 5              │
│                                                      │
│  ┌──────────────────────────────┬──────────────┐   │
│  │  📋 Копировать               │   🗑️         │   │
│  └──────────────────────────────┴──────────────┘   │
│                                                      │
│  ┌────────────────────────────────────────────┐    │
│  │         ▼ Развернуть                       │    │
│  └────────────────────────────────────────────┘    │
│                                                      │
└──────────────────────────────────────────────────────┘
```

**Пояснения:**
- Заголовок крупным шрифтом
- Метаданные: дата, время, количество сцен
- Первый ряд кнопок: "Копировать" (широкая) + "Удалить" (узкая, иконка)
- Второй ряд: кнопка "Развернуть" на всю ширину

## 🎯 Интерактивные состояния

### Кнопка "Копировать"

```
Обычное:      [  📋 Копировать  ]
Наведение:    [  📋 Копировать  ]  (фон: gray-50)
После клика:  [  ✓ Скопировано! ]  (на 2 сек)
                   ↓
               [Toast внизу]
               "Сценарий скопирован
                в буфер обмена"
```

### Кнопка "Развернуть/Свернуть"

```
Свёрнуто:  [     ▼ Развернуть     ]  (outline, белый фон)
           ↓ клик
           [Animation: expand]
           ↓
Развёрнуто: [     ▲ Свернуть     ]  (default, синий фон)
```

### Кнопка "Удалить"

```
Обычное:  [ 🗑️ ]
           ↓ клик
         ┌─────────────────────┐
         │ Удалить сценарий?   │
         │                     │
         │ Сценарий "..." будет│
         │ удалён безвозвратно │
         │                     │
         │ [Отмена] [Удалить]  │
         └─────────────────────┘
                    ↓ подтверждение
                 [Toast]
                 "Сценарий удалён"
                    +
              Карточка исчезает
```

## 🔄 Последовательность действий

### Сценарий 1: Просмотр сценария

```
1. Открыть "Сохранённые" (нижняя панель)
   ↓
2. Скролл списка сценариев
   ↓
3. Клик "▼ Развернуть" на нужном
   ↓
4. Карточка расширяется (анимация)
   ↓
5. Прокрутка содержимого (если не влезает)
   ↓
6. Клик "▲ Свернуть"
   ↓
7. Карточка сворачивается
```

### Сценарий 2: Копирование сценария

```
1. Найти нужный сценарий
   ↓
2. Клик "📋 Копировать"
   ↓
3. Текст скопирован в буфер
   ↓
4. Toast: "Скопировано!"
   ↓
5. Вставить в другое приложение (Ctrl+V)
```

### Сценарий 3: Удаление сценария

```
1. Найти ненужный сценарий
   ↓
2. Клик на иконку "🗑️"
   ↓
3. Диалог подтверждения
   ↓
4. Клик "Удалить"
   ↓
5. Toast: "Сценарий удалён"
   ↓
6. Карточка исчезает (анимация)
   ↓
7. Badge в панели обновляется (-1)
```

## 📏 Размеры и отступы

### Карточка
- Padding: `p-4` (мобильные) / `md:p-6` (десктоп)
- Border radius: `rounded-xl`
- Shadow: `shadow-sm`
- Gap между элементами: `space-y-4`

### Кнопки
- Высота: `size="sm"` (≈ 36px)
- Отступ между: `gap-2` (8px)
- Иконка: `h-4 w-4` (16x16px)

### Номера сцен
- Мобильные: `w-12 h-12` (48x48px)
- Десктоп: `md:w-16 md:h-16` (64x64px)
- Border radius: `rounded-lg`
- Градиент: `from-indigo-500 to-purple-600`

### ScrollArea
- Мобильные: `max-h-[70vh]`
- Десктоп: `md:max-h-[600px]`
- Padding внутри: `p-4 md:p-6`

### Страница
- Padding внизу: `pb-24` (для нижней панели)
- Max width: `max-w-5xl`
- Высота списка: `h-[calc(100vh-300px)]`
