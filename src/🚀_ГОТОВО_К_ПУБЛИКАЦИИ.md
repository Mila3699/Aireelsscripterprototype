## 🎯 Что это?

**AI Reels Scripter** — веб-приложение для анализа коротких видео с помощью Google Gemini AI.

### Ключевые особенности:

✅ **Drag & Drop загрузка** видео до 100 МБ  
✅ **Mock-данные** — 3-секундная имитация анализа  
✅ **4 блока результатов** — транскрибация, ключи к успеху, сценарий, рекомендации  
✅ **Celebration анимация** с конфетти (только при первом результате)  
✅ **Копирование** результатов в буфер обмена  
✅ **LocalStorage** — сохранение сценариев локально  
✅ **Аутентификация** через mock-систему (email/password)  
✅ **Responsive дизайн** — адаптивен для мобильных  
✅ **Apple HIG стиль** — чистый минималистичный интерфейс

## 🔒 Безопасность (Mock-система)

### ⚠️ Важно: Это ПРОТОТИП!

Текущая реализация использует **mock-данные**:

```javascript
// lib/supabase.ts
export const mockUsers = [
  { 
    email: 'demo@example.com', 
    password: 'demo123', 
    name: 'Demo User' 
  }
];

// Аутентификация через localStorage:
localStorage.setItem('mockAuth', JSON.stringify(user));
```

### ⚠️ НЕ использовать в production!

Для реального production приложения нужно:
1. Настроить Supabase Auth
2. Подключить Google Gemini API
3. Настроить storage для видео
4. Добавить rate limiting
5. Настроить CORS и HTTPS

## 🎉 Celebration система

### Как работает:

```typescript
// Показывается только при ПЕРВОМ получении результатов
const hasSeenCelebration = localStorage.getItem('hasSeenCelebration');

if (!hasSeenCelebration && analysisResult) {
  // 🎊 Запуск конфетти!
  confetti({ ... });
  
  // Сохраняем флаг
  localStorage.setItem('hasSeenCelebration', 'true');
}
```

### Параметры конфетти:
- ✅ Золотые и радужные цвета
- ✅ 3 залпа (150-250-100 частиц)
- ✅ Разные углы (25°, 90°, 155°)
- ✅ Автоматическая очистка canvas

## 🆘 Troubleshooting

### Проблема: Конфетти не появляются

**Решение:**
```javascript
// В консоли браузера:
localStorage.removeItem('hasSeenCelebration');
location.reload();
// Теперь загрузите видео снова
```

---

### Проблема: Кнопка "Выйти" не работает (в preview)

**Решение:**
```javascript
// В консоли браузера:
localStorage.removeItem('mockAuth');
location.reload();
```

**ИЛИ** дождитесь публикации — там должна работать.

---

### Проблема: Сохранённые сценарии исчезли

**Причина:** localStorage очищен или режим инкогнито

**Решение:** Используйте обычный режим браузера (не инкогнито)

---

### Проблема: "Анализировать" неактивна

**Причина:** Файл не прошёл валидацию

**Проверьте:**
- ✅ Формат: MP4, MOV, WebM
- ✅ Размер: < 100 МБ
- ✅ Файл не повреждён
