## ⚙️ Что происходит "под капотом"

### 1. Пользователь загружает видео:

```
Пользователь → Drag & Drop видео
              ↓
React получает файл
              ↓
Проверка размера (макс 100 МБ)
              ↓
✅ Файл принят → Показать превью
```

### 2. Нажимает "Анализировать":

```
Клик на кнопку
      ↓
Показать экран загрузки (Processing)
      ↓
Ждать 3 секунды (setTimeout)
      ↓
Загрузить mock-данные из /lib/mockData.ts
      ↓
Показать результат + конфетти 🎉
```

### 3. Сохраняет результат:

```
Клик на "Сохранить"
      ↓
Отправить в Supabase Database
      ↓
localStorage.setItem() (резервная копия)
      ↓
Показать toast: "Сохранено!" ✅
```

### 4. Просматривает сохранённые:

```
Клик на "Сохранённые"
      ↓
Запрос к Supabase Database
      ↓
Получить массив сохранённых сценариев
      ↓
Отобразить список
```

## 🔄 Обновление после публикации

### Netlify (ручное):

```
1. Внесите изменения в файлы на компьютере
2. Откройте Netlify Dashboard
3. Перетащите новую папку (Deploys → Drag and drop)
4. Подождите 2 минуты
5. Готово! Сайт обновлён ✅
```

### Vercel (автоматическое):

```
1. Внесите изменения в файлы
2. Закоммитьте в GitHub (git commit + git push)
3. Vercel автоматически пересоберёт
4. Готово! ✅
```
