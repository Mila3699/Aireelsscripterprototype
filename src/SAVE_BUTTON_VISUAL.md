## ✨ Детали анимаций

### Появление блока:

```css
initial: { opacity: 0, y: 20 }
animate: { opacity: 1, y: 0 }
transition: { delay: 3.5 + script.length * 0.1 }
```

```
      Появление
         
    t=0s    t=0.3s    t=0.6s
     ↓        ↓         ↓
    ░░░     ▒▒▒▒     ████
    
    Opacity: 0 → 0.5 → 1
    Y: +20px → +10px → 0px
```

### Пульсация кнопки:

```css
animate: { scale: [1, 1.02, 1] }
transition: { duration: 2, repeat: Infinity }
```

```
    Scale timeline
    
    1.02 ┤     ╱╲     ╱╲
         │    ╱  ╲   ╱  ╲
    1.00 ┤───╯    ╲─╯    ╲─
         └─────────────────→
         0s  1s  2s  3s  4s
         
    Бесконечный цикл
```

### Иконка Heart (spring):

```css
initial: { scale: 0 }
animate: { scale: 1 }
transition: { type: "spring", stiffness: 200 }
```

```
    Spring animation
    
    1.2 ┤      ╱╲
        │     ╱  ╲╱╲
    1.0 ┤────╯      ╲___
        │              
    0.0 ┤─
        └─────────────→
        0s  0.2s  0.4s
        
    "Отскок" эффект
```

## 🏆 Лучшие практики

### ✅ DO:

- Делайте кнопку заметной (большой размер)
- Используйте призыв к действию (CTA)
- Объясняйте ценность (зачем сохранять)
- Добавляйте анимации (привлекает внимание)
- Давайте положительное подкрепление (конфетти, сердце)

### ❌ DON'T:

- Не делайте слишком навязчиво
- Не спамьте множеством кнопок
- Не используйте aggressive язык
- Не забывайте про accessibility
- Не игнорируйте mobile версию
