## 2️⃣ БЫСТРО (10 минут) ⭐ РЕКОМЕНДУЕТСЯ

### Через Netlify:

```
1. Скачать проект (ZIP)
2. Распаковать
3. Зайти на netlify.com
4. Зарегистрироваться (через GitHub)
5. "Add new site" → "Deploy manually"
6. Перетащить папку
7. Подождать 2 минуты
8. Скопировать ссылку
9. Настроить Supabase (добавить URL)
10. ГОТОВО!
```

**Результат:** https://ваш-сайт.netlify.app  
**Время:** 10 минут  
**Стоимость:** Бесплатно  
**Навсегда:** ✅

### Подробная инструкция:
```
→ /ШПАРГАЛКА_ПУБЛИКАЦИЯ.md
```

## 3️⃣ ПРОФЕССИОНАЛЬНО (15 минут)

### Через Vercel + GitHub:

```
1. Создать аккаунт GitHub
2. Создать репозиторий
3. Загрузить файлы
4. Зайти на vercel.com
5. Зарегистрироваться (через GitHub)
6. Import Project
7. Deploy
8. Настроить Supabase
9. ГОТОВО!
```

**Результат:** https://ваш-сайт.vercel.app  
**Плюсы:** Автообновления при изменении кода  
**Время:** 15 минут

## ⚙️ ОБЯЗАТЕЛЬНО: Настройка Supabase

После публикации на Netlify/Vercel:

```
1. Откройте:
   https://supabase.com/dashboard/project/ssqcxrimivxqdydgmfcn/settings/auth

2. Добавьте в Site URL:
   https://ваш-сайт.netlify.app

3. Добавьте в Redirect URLs:
   https://ваш-сайт.netlify.app/**

4. Отключите "Enable email confirmations"

5. Нажмите Save
```

**Без этого регистрация НЕ будет работать!** ⚠️

## 🆘 ЧТО-ТО НЕ РАБОТАЕТ?

### Ошибка входа:
```
→ Отключили "Email confirmations"?
→ Добавили URL в Supabase?
→ Смотрите: /ОШИБКА_ВХОДА_РЕШЕНИЕ.md
```

### Приложение не загружается:
```
→ Сборка завершена?
→ Ctrl + F5 (обновить)
→ Проверьте консоль (F12)
```

### Не работает сохранение:
```
→ Supabase настроен?
→ Пользователь вошёл?
→ Проверьте статус: status.supabase.com
```
