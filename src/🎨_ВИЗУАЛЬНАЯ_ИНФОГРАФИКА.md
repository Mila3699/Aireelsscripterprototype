## 💾 ПОТОК ДАННЫХ

```
РЕГИСТРАЦИЯ:
───────────
1. Пользователь → Email + Пароль
2. React → Отправка в Supabase Auth
3. Supabase → Создание аккаунта
4. Supabase → Возврат токена
5. React → Сохранение в localStorage
6. React → Перенаправление на главную

ЗАГРУЗКА ВИДЕО:
──────────────
1. Пользователь → Drag & Drop файла
2. React → Валидация (размер, формат)
3. React → Показ превью
4. React → Сохранение в state

АНАЛИЗ (ПРОТОТИП):
──────────────────
1. Клик "Анализировать"
2. React → Показ экрана загрузки
3. React → setTimeout(3000) ← ИМИТАЦИЯ
4. React → Загрузка mock-данных
5. React → Показ результата + конфетти

СОХРАНЕНИЕ:
───────────
1. Клик "Сохранить"
2. React → Отправка в Supabase Database
3. Supabase → Сохранение в таблице
4. React → localStorage (резервная копия)
5. React → Toast "Сохранено!" ✅

ПРОСМОТР СОХРАНЁННЫХ:
─────────────────────
1. Клик на "Сохранённые"
2. React → Запрос к Supabase
3. Supabase → Возврат массива
4. React → Отображение списка
```
